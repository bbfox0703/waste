SIMD (Single Instruction, Multiple Data): These instructions allow operations to be performed on multiple data elements simultaneously, which is particularly useful for vectorized computations.

AVX2 (Advanced Vector Extensions 2): These are extended versions of SIMD instructions with VEX encoding, allowing for greater performance and flexibility, such as wider registers and improved parallelism. For example, `addss` is extended to `vaddss` in AVX2, supporting additional features like non-destructive three-operand forms.
